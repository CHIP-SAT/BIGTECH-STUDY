
# 컴퓨터가 이해하는 정보

> 프로그램 개발 → 프로그래밍 → 자바, 파이썬, C++

컴퓨터는 프로그래밍 언어를 직접 이해하지 못함 → 데이터, 명령어
> 
- 데이터
    - 숫자, 문자, 이미지, 동영상
    - 컴퓨터와 주고받는 정보나 컴퓨터에 저장된 정보 자체
- CPU
    - 명령어를 실행하는 주체
    - CPU 종류에 따라 실행 가능한 세부적인 명령어의 종류와 처리 양상이 달라질 수 있음

# 컴퓨터의 핵심 부품

## CPU

> 데이터와 명령어를 읽고, 해석하고, 실행하는 부품
> 
- 산술논리연산장치
    - 사칙 연산, 논리 연산같은 연산을 수행할 회로
    - 실질적인 연산
- 제어장치
    - 명령어를 해석해 제어 신호라는 전기 신호를 내보내는 장치
    - 부품을 작동시키기 위한 신호
- 레지스터
    - 데이터와 명령어를 처리하는 과정의 중간값

## 메모리와 캐시 메모리

- 메인 메모리
    - ram
    - cpu가 읽고, 해석하고, 실행하는 모든 정보
    - 실행 중인 프로그램을 구성하는 데이터와 명령어를 저장하는 부품
- 주소
    - CPU가 메모리에 접근할 때 주소가 있어야 빠르게 접근 가능
    - CPU가 원하는 정보로 접근하기 위한 정보
- 휘발성
    - 저장되고 있는 정보가 지워질 수 있음
- 캐시 메모리
    - CPU가 더 빨리 메모리에 저장된 값에 접근하기 위해 사용하는 장치

## 보조기억장치

- 비휘발성
- CD-ROM, DVD, HDD, SSD
- 보관할 프로그램을 저장

## 입출력장치

- 컴퓨터 외부에 연결되어 컴퓨터 내부와 정보를 교환하는 장치
- 입력장치, 출력장치

## 메인 보드와 버스

- 메인보드
    - 컴퓨터의 핵심 부품을 비롯한 여러 부품들을 연결할 수 있는 슬롯
- 버스
    - 각 컴퓨터 부품들이 정보를 주고받는 통로
    - 시스템 버스
